Anni Albers

Dit is een digitale reproductie van het werk van Anni Albers.

Wanneer je meerdere varianten genereert, dan valt je op dat de gegenereerde patronen minder goed lijken op het origineel. Zo is er in het orgineel bijvoorbeeld een ritme/patroon in het aantal lichtgekleurde vlakken. Om de code echter nog een beetje overzichtelijk te houden, heb ik ervoor gekozen om dit niet in de code te verwerken. Voel jezelf echter vrij om dit zelf toe te voegen aan de code.

Voor meer uitleg over hoe deze digitale reproductie werkt, zie: https://github.com/JeffreyArts/p5-anni-albers
