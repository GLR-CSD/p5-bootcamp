Het regent karakters!

Selecteer het canvas door op het witte vlak te tekenen. Druk daarna op de toetsen om de karakters naar beneden te laten vallen. Er zit ook een stukje code in die er voor zorgt dat de array met karakters wordt opgeschoond door karakters die buiten het scherm zijn gevallen automatisch worden verwijderd.

Voor meer uitleg over de keyPressed functie, zie: https://p5js.org/reference/p5/keyPressed

Om deze demo te laten werken, maken we gebruik van een array waar we alle karakters die je intypt in opslaan. Vervolgens loopen we door deze array en updaten we de y postitie zodat ze omlaag vallen. Zodra ze buiten het scherm zijn gevallen, verwijderen we ze.
