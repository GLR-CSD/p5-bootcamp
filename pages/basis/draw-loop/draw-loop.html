Basis - Draw loop

De code in de draw functie wordt ieder frame uitgevoerd, door hier slim mee om te gaan kun je leuke animaties of toffe games maken!

Voor meer uitleg over de draw functie, zie: https://p5js.org/reference/p5/draw

Als dit de eerste keer is dat je met P5 werkt, check dan ook even hoe je de P5 library kunt downloaden, zodat je het kunt inladen via de script tag.
